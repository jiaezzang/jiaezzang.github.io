# 브랜치 전환하기

## 전환하고자 하는 브런치 생성

```
git checkout -b [새 브런치]
```

<br />

## 브런치 전환

```
git checkout [전환하고자 하는 브런치]
```

<br />

## 이전 브런치로 돌아가기

```
git checkout -
```
