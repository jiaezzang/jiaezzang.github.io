# 3. [React 패키지 다운로드](https://crate-react-app.dev)

![img](https://img1.daumcdn.net/thumb/R1280x0/?scode=mtistory2&fname=https%3A%2F%2Fk.kakaocdn.net%2Fdn%2Fb70MB4%2FbtrQE5iueL8%2FMIboADp7J60BD8KkRmRC2k%2Fimg.png)

```

```
